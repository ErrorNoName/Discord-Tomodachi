## C’est quoi ?

**Discord Tomodachi** est une appli bureau Windows qui transforme tes amis Discord en habitants d’une île style *Tomodachi Life*.

| | |
|---|---|
| Connexion | Token Discord (ton compte) |
| Monde | Île, saisons, nuit, déco |
| Social | Amis, chambres, bonheur, messages |
| IA | Répond dans **ton** style (données locales) |
| Jeu | Café, mini-jeux, quêtes, studio dessin |
| Accès | Rôle Discord **Licencié** (payant) |

> Non affilié à Discord ni à Nintendo.

## Accès payant (licence Discord)

<p align="center">
  <img src="assets/icon-heart-full.png" width="56" alt="Licence" />
</p>

Le logiciel est **payant**. Au login, l’app vérifie avec **ton** Discord que :

1. tu es sur le serveur officiel **Discord Tomodachi**
2. tu as le rôle **Licencié**

Sinon → popup « Rejoindre le Discord / Demander l’accès ».

| | |
|---|---|
| Invite | **[discord.gg/mVDVkJ4Zvz](https://discord.gg/mVDVkJ4Zvz)** |
| Salon | `#obtenir-accès` |
| Rôle | **Licencié** |

```env
LICENSE_ENABLED=1
LICENSE_GUILD_ID=...
LICENSE_ROLE_ID=...
LICENSE_INVITE_URL=https://discord.gg/mVDVkJ4Zvz
LICENSE_ACCESS_CHANNEL_ID=...
```

Dev sans licence : `LICENSE_ENABLED=0`.

Setup serveur (rôles, salons, perms, invite) :

```bat
python tools\setup_license_discord_server.py
python tools\post_license_webhook_embeds.py
```

Docs : [`docs/discord-server/README.md`](docs/discord-server/README.md)

## Lancer (développement)

### Prérequis

- Windows 10 / 11  
- Node.js 18+  
- Python 3.11+ (ou Python embarqué dans `game/resources/backend`)

### En un clic

```bat
Tomodachi.bat
```

Build UI + lance Electron + backend.

### À la main

```bat
cd game
npm install
npm run build
npx electron .
```

Backend seul :

```bat
set PYTHONPATH=%CD%
python -m server
```

→ `http://127.0.0.1:8787`

### Configuration (`.env`)

Copie `.env.example` → `.env` :

| Clé | Description |
|-----|-------------|
| `TOKEN` | Token Discord (aussi via l’UI) |
| `CURSOR_API_KEY` | Clé IA (agent Cursor) |
| `ELEVENLABS_API_KEY` | Optionnel — SFX / voix |
| `LICENSE_*` | Serveur + rôle payant |

## Structure du projet

```
Discord-Tomodachi/
├── assets/                 # Art du jeu (logo, île, icônes, screens…)
├── Tomodachi.bat           # Lanceur dev
├── config.py               # Config centrale
├── server/                 # FastAPI — API jeu + licence
├── game/                   # Electron + React
├── harvester/              # Discord REST (token user)
├── ai/                     # Agent, personas, relations
├── storage/                # SQLite / vecteurs
├── tools/                  # Setup Discord, embeds, tests
├── site/                   # Landing GitHub Pages
├── docs/                   # Packaging, audio, serveur licence
└── data/                   # Runtime local (gitignored)
```

## Sécurité

- Ne **jamais** partager ton token Discord ni commit `.env`
- Usage d’un token utilisateur = zone grise / risque vs [ToS Discord](https://discord.com/terms)
- Conversations & base restent **locales** ; l’IA dépend de ta `CURSOR_API_KEY`
