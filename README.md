# Landing Discord Tomodachi

Page d’accueil animée pour GitHub Pages.

## Contenu

- Splash (logo + floaties, sans barre de chargement)
- Curseur main `tour-point-hand`
- Galerie d’aperçus (scène pleine largeur + miniatures)
- Boutons avec effet poussoir
- Police DreamLife, thème couleurs du jeu

## Publier

GitHub → **Settings → Pages** → branch `main`, folder **`/site`**.

## Local

Ouvre `index.html`, ou :

```bat
npx --yes serve .
```

Le README complet du projet est à la racine : [`../README.md`](../README.md).
